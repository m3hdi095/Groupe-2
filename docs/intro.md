## Qu'est-ce que Git ?

Git est un système de gestion de versions distribué qui permet de suivre l'évolution d'un projet informatique. Il facilite la collaboration entre développeurs en permettant de travailler simultanément sur le même code.

### Les principes clés de Git :

- **Historique complet** : Git conserve toutes les versions des fichiers et permet de revenir en arrière à tout moment
- **Travail en branches** : Chaque développeur peut travailler sur sa propre branche sans affecter le code principal
- **Collaboration efficace** : Git permet de fusionner facilement les modifications de plusieurs personnes
